# Number Guessing Game (Java)

This is a simple Java mini project created for learning purposes.

## Features
- Generates a random number between 0 and 99
- User guesses the number
- Provides hints (too large / too small)

## Concepts Used
- Scanner
- do-while loop
- if-else condition
- Math.random()

## How to Run
javac main16.java
java main16
